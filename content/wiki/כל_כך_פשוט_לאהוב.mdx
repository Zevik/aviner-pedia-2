## כל כך פשוט לאהוב

\[מ"באהבה ובאמונה"\]

כל כך פשוט לאהוב, לחייך אל החיים, לשמוע את שירת החיים, לתת להם ללטף
אותנו, לתת ללבותינו לפתוח את החלון אל שמש האהבה שהופכת אותנו לטובים
יותר.

נאהב את ד', נאהב את עם ישראל, את כל האדם באשר הוא אדם, כל יצור, כל בעל
חיים, וכל גרגר חול, כל צומח וכל פרח. נאהב את כל התורה, את כל המצווה, כל
מנהג, כל הלכה, כל אות, כל מידה טובה. נאהב את ארצנו, כל עיר, כל יישוב, כל
הר, כל גבעה. נאהב את האהבה, נאהב את היראה, נאהב את הרצינות, נאהב את
השמחה. נאהב, נאהב. כל כך פשוט לאהוב. זה כל כך טבעי.

נאהב את האוויר הצח, נאהב את תכול השמים, את הנשמה מלאת התקווה, את האומץ
ואת הגבורה. נאהב את המים ואת העפר, את הגשם ואת השלג.

נאהב לומר בבקשה, נאהב לומר תודה. נאהב להיות טובים, נאהב להיות עדינים,
נאהב כל מה שראינו, כל מה ששמענו, כל מה שחווינו. את העבר, את ההווה ואת
העתיד. את הצעירים ואת הזקנים, את הילדות ואת הילדים. נאהב. נאהב. זה כל כך
פשוט.

נאהב את הנשמה המלאה, כל כך מלאה. נאהב את רגעינו הטובים ואת רגעינו הקשים.
נאהב להיות בני בית, נאהב גם כשאנו זרים. נאהב את העולם הזה, נאהב את העולם
הבא.

נאהב את דרכנו בעולמנו, נאהב לחצות את הנהר אל עבר עולם אחר. נאהב לפגוש את
ריבונו של עולם כאן, נאהב לפוגשו שם. כל כך פשוט לאהוב.

נאהב לנוח שם, לקבל את שכר מאמצינו שם, נאהב לשיר עם המלאכים שם.

נאהב לעמול פה, נאהב להחזיק מעמד בגופנו פה, נאהב לעבוד את ד' פה. כל כך
פשוט.

נאהב את ד' פה, נאהב את ד' שם, נאהב את ד' בכל מקום. נאהב לאהוב. לאהוב.
