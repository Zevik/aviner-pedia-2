## בעיות ידועות ופתרונות

### בעיה: תמונות YouTube לא נטענות
**סיבה אפשרית:** YouTube חוסם תמונות ממוזערות מסוימות
**פתרון:** השתמש ב-mqdefault.jpg או hqdefault.jpg

### בעיה: Markdown לא מרונדר נכון
**סיבה אפשרית:** תווים מיוחדים ב-CSV
**פתרון:** בדוק את התוכן ב-Supabase, וודא ניקוי נכון בעת seed

### בעיה: חיפוש לא עובד
**סיבה אפשרית:** FTS index לא נוצר
**פתרון:** הפעל שוב את הפקודה CREATE INDEX מ-schema.sql
